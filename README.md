Aplicação de lista de tarefas criada para praticar lógica de programação, eventos e manipulação do DOM, com foco em organização e experiência do usuário.

## 🚀 Tecnologias
- HTML
- CSS
- JavaScript

## 🎯 O que pratiquei
- Eventos com JavaScript
- Manipulação do DOM
- Organização de listas
- Interação com o usuário
- Estruturação de interface
